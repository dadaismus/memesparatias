# README

Memesparatias.com, está pensado para esa tía que todos llevamos dentro, la plataforma está hecha en Ruby on Rails con la finalidad de aprender, si gustas contruibuir, gracias!.

* Diseño del buscador

* Diseño de cards

* Compartir vía Whatsapp
